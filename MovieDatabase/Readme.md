# 🎬 Movie Database (C++ / SQLite)

## 📌 Описание

**Movie Database** — это консольное приложение на C++, реализующее упрощённую базу данных фильмов с использованием SQLite.

Программа демонстрирует полный цикл работы с данными:

* создание базы данных
* хранение информации
* выполнение SQL-запросов
* агрегацию данных в C++

---

## 🚀 Возможности

* 📀 Создание и инициализация SQLite базы данных
* 🎬 Добавление фильмов
* 👥 Добавление актёров
* 🎭 Связывание актёров с фильмами (casting)
* 🖼️ Хранение медиа (BLOB)
* 🔎 Получение фильмов с актёрским составом (`get_movies()`)
* 🔗 Использование JOIN-запросов

---

## 🧩 Структура базы данных

Программа использует нормализованную схему:

```text
movies
  ↓
casting
  ↓
persons
```

---

### 📂 Таблицы

#### 🎬 movies

* `id` — идентификатор
* `title` — название
* `year` — год выпуска
* `length` — длительность

#### 👥 persons

* `id` — идентификатор
* `name` — имя

#### 🎭 casting

* `movieid` — фильм
* `personid` — актёр
* `role` — роль

#### 🖼️ media

* `id` — идентификатор
* `movieid` — фильм
* `name` — имя файла
* `text` — описание
* `blob` — бинарные данные

---

## ⚙️ Основной функционал

### 1. Инициализация базы

* создание файла базы данных
* создание таблиц

---

### 2. Заполнение данными

* добавление фильма
* добавление актёров
* привязка ролей
* добавление медиа

---

### 3. Получение данных (`get_movies()`)

* выполняется один SQL-запрос с `JOIN`
* данные агрегируются в структуру `movie`

```text
movie
 ├── id, title, year, length
 └── cast[]
```

---

### 4. Вывод результатов

Пример:

```text
1 | Мир Юрского периода: Возрождение (2025) 134 min
   - Scarlett Johansson as Zora Bennett
   - Jonathan Bailey as Dr. Henry Loomis

2 | Как приручить дракона (фильм) (2025) 116 min
   - Mason Thames as Hiccup
   - Nico Parker as Astrid

3 | Аватар — Пламя и пепел (2025) 192 min
   - Sam Worthington as Jake Sully
   - Zoe Saldaña as Neytiri
```
