# 🎣 Игра «Ловля рыбы» (C++)

Консольная игра-симуляция, в которой игрок пытается поймать рыбу, выбирая сектор пруда.

---

## 📌 Описание

Игровое поле представляет собой **9 секторов пруда**.

В начале игры случайным образом размещаются:

* 🐟 **1 рыба**
* 👢 **3 сапога**

Каждый объект занимает **уникальный сектор** (без пересечений).

Игрок вводит номер сектора (от 0 до 8) и пытается поймать рыбу.

---

## 🎮 Правила игры

* Если игрок выбирает **пустой сектор** → попытка продолжается
* Если игрок находит **рыбу** → 🎉 победа
* Если игрок находит **сапог** → ❌ проигрыш

---

## ⚙️ Особенности реализации

* Используется генерация случайных чисел:

  ```cpp
  std::srand(std::time(nullptr));
  ```

* Размещение объектов:

  ```cpp
  field[std::rand() % 9]
  ```

* Применяются **исключения (exceptions)**:

  * 🐟 Рыба — пользовательское исключение (успех)
  * 👢 Сапог — пользовательское исключение (проигрыш)

---

## 🧠 Логика работы

1. Создаётся массив из 9 секторов

2. Случайно размещаются:

   * 1 рыба
   * 3 сапога

3. Игрок вводит номер сектора

4. Проверка:

   * Рыба → выбрасывается исключение (победа)
   * Сапог → выбрасывается исключение (проигрыш)
   * Пусто → игра продолжается

---

## 📊 Результат

После завершения игры выводится:

* Успех или поражение
* Количество попыток
