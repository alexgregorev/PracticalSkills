# Mini Search Engine (C++)

Небольшая реализация **поискового движка** на C++ (80 строк кода).
Проект показывает базовые идеи, которые используются в реальных поисковых системах:

* индексирование документов
* inverted index
* подсчёт совпадений (ranking)
* сортировка результатов поиска

Код предназначен для **учебных целей** и демонстрирует, как можно реализовать простую поисковую систему на стандартных структурах C++.

---

# Идея

Поисковик строит **обратный индекс (inverted index)**.

Вместо хранения:

```
document → words
```

мы храним:

```
word → documents
```

Пример:

```
apple
   1 → 2
   3 → 3

banana
   1 → 1
   2 → 1
```

Где:

* число слева — `doc_id`
* число справа — **частота слова в документе**

---

# Алгоритм работы

### 1. Добавление документа

Документ разбивается на слова.

Каждое слово добавляется в индекс:

```
word → (document, frequency++)
```

---

### 2. Поиск

Запрос может содержать несколько слов:

```
apple banana
```

Алгоритм:

1. Для каждого слова запроса ищем документы в индексе
2. Суммируем частоты слов для каждого документа
3. Сортируем документы по score

```
score = сумма совпадений слов
```

---

# Структуры данных

Основная структура:

```cpp
unordered_map<string, unordered_map<int,int>> index;
```

Что означает:

```
word → (doc_id → frequency)
```

```

   ↓

   ↓

   ↓

```

---

# Сложность

Добавление документа

```
O(words)
```

Поиск

```
O(matches log N)
```

где:

* `matches` — количество найденных документов
* `N` — количество результатов
