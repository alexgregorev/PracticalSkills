# Cosine Similarity (C++)

Программа вычисляет **степень сходства двух текстов** с помощью алгоритма **Cosine Similarity**.  
Метод широко используется в системах поиска, анализе текстов и NLP.

## Идея алгоритма

Каждый текст представляется в виде **вектора частот слов**.

Например:

Text 1

machine learning improves search


Text 2

search engines use machine learning


После подсчёта слов получаются векторы:

| word | Text 1 | Text 2 |
|-----|------|------|
machine | 1 | 1
learning | 1 | 1
improves | 1 | 0
search | 1 | 1
engines | 0 | 1
use | 0 | 1

Затем вычисляется угол между векторами.

## Формула

## 📐 Cosine Similarity

**cos(θ) = (A · B) / (||A|| * ||B||)**

- **A · B** — скалярное произведение  
- **||A||, ||B||** — длины векторов  

Результат находится в диапазоне:


1 → тексты одинаковые

0 → тексты полностью разные


## Как работает программа

Программа выполняет несколько шагов:

1. **Tokenize**  
   Разбивает текст на слова.

2. **Word frequency**  
   Подсчитывает количество вхождений каждого слова.

3. **Vector creation**  
   Формирует вектор частот слов.

4. **Cosine similarity**  
   Вычисляет сходство двух текстов.

## Пример результата


Text 1: machine learning improves search

Text 2: search engines use machine learning

Similarity = 0.67


Это означает, что тексты **достаточно похожи**.

## Где используется

Cosine Similarity применяется в:

- поисковых системах  
- поиске похожих документов  
- рекомендательных системах  
- кластеризации текстов  
- анализе данных  

## Структура программы


tokenize() → разбивает текст на слова

word_freq() → считает частоты слов

cosine_similarity() → вычисляет сходство

main() → демонстрация работы
