# 🎬 Video Player (Qt / QML)

## 📌 Описание

**Video Player** — это графическое приложение видеопроигрывателя, разработанное с использованием **Qt (QML)**.

Приложение реализует полноценный пользовательский интерфейс и базовую логику воспроизведения видео, включая управление проигрыванием и синхронизацию состояния с интерфейсом.

---

## 🚀 Возможности

* ▶️ Воспроизведение видео
* ⏸️ Пауза
* ⏹️ Остановка
* ⏪ Перемотка назад
* ⏩ Перемотка вперёд
* 📊 Индикатор прогресса воспроизведения
* 🎞️ Отображение видео в реальном времени

---

## 🧩 Архитектура интерфейса

Интерфейс приложения состоит из двух основных частей:

### 🎬 Область видео

* Компонент `VideoOutput`
* Отображает воспроизводимое видео

### 🎛️ Панель управления

* Кнопки управления:

  * ▶️ Play
  * ⏸️ Pause
  * ⏹️ Stop
  * ⏪ Rewind
  * ⏩ Forward
* Ползунок прогресса (`Slider` / `ProgressBar`)

---

## ⚙️ Технологии

* **Qt 6**
* **QML (Qt Quick)**
* **Qt Multimedia**

  * `MediaPlayer`
  * `VideoOutput`

---

## 🔁 Принцип работы

Приложение использует `MediaPlayer` для управления воспроизведением:

```text
MediaPlayer → VideoOutput → UI
```

* `MediaPlayer` загружает и воспроизводит видео
* `VideoOutput` отображает видеопоток
* UI-компоненты управляют состоянием плеера
