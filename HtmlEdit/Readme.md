# 🧩 HTML Editor (Qt)

Приложение для редактирования HTML-кода с мгновенным отображением результата.

---

## 📌 Описание

Данное приложение представляет собой простой HTML-редактор, реализованный с использованием Qt.
Окно разделено на две равные части:

* **Слева** — редактор HTML-кода
* **Справа** — визуализация страницы (рендеринг)

При изменении текста HTML-документа результат автоматически обновляется в режиме реального времени.

---

## 🖥️ Интерфейс

```
+----------------------+----------------------+
|                      |                      |
|   HTML Editor        |   Web View           |
|   (QPlainTextEdit)   |   (QWebEngineView)   |
|                      |                      |
+----------------------+----------------------+
```

---

## ⚙️ Используемые компоненты

* QPlainTextEdit — ввод и редактирование HTML
* QWebEngineView — отображение HTML
* QHBoxLayout — горизонтальное расположение виджетов
* QSizePolicy — управление размерами элементов

---

## 🚀 Функциональность

* Редактирование HTML-кода
* Мгновенное обновление отображения
* Адаптивное изменение размеров окна
* Разделение интерфейса 50/50

---

## 🔄 Обновление HTML в реальном времени

```cpp
connect(editor, &QPlainTextEdit::textChanged, [=]() {
    view->setHtml(editor->toPlainText());
});
```
